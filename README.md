## Installation

To install and run this project locally, follow these steps:

1. Clone the repository:

 ```bash
   git clone https://github.com/MarVik-Dev/marvik-portfolio.git
   ```

2. Rename and access the project directory:
   cd your-portfolio

3. Install the dependencies (at your convenience):
   npm install
   or
   pnpm install

4. Run the project:
   npm run dev
   or
   pnpm dev

5. Open your browser:
   Access http://localhost:3000 to see your new portfolio in action.
